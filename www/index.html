🚀 webserver

基于 Reactor 模式的 C++11 高性能 Web 服务器

✨ 特性

  • Multi-Reactor 多线程架构 (主线程 Accept + 子线程 IO)
  • Epoll + 非阻塞 IO 事件驱动
  • HTTP/1.1 Keep-Alive 长连接支持
  • 哈希分片 LFU 静态文件缓存
  • 内存池优化 per-connection 分配
  • 异步日志系统

📊 技术栈

事件模型 Epoll (ET/LT)
线程模型 One Loop Per Thread
缓存策略 LFU + 哈希分片
内存管理 固定大小槽位内存池
日志 双缓冲异步日志
协议 HTTP/1.1
